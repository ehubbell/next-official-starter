## Overview
The official starter for the popular NextJS framework.


## Prerequisites
- Playbooks
- Node
- NPM
- NextJS


## Quick Start
- playbooks install next-official-starter
- cd next-official-starter
- npm install
- npm run dev
- visit http://localhost:3000
